Turning Five Nights at Freddy’s into a tabletop board game.

We chose this game to be turned into a board game because the game has appealing aesthetic and unique gameplay. I designed the maps for this board game in pixel arts because they fit the 2D theme. This will be a 2-player game. Since, in the original game, the guard does not accurately know the positions of the animatronics, there will be 2 maps, one for the animatronics’ view, and the other for the security guard’s view.

Map difference: Animatronics' map have tiles that indicate squares that the animatronics can move on. The security guard's map does not have tiles because, in the game, the security guard does not know how long it will take for the animatronics to move.
