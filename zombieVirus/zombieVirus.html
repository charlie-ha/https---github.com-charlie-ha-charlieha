Horror Game Project

Project highlight:

Project type:

First Person Horror Survival Game in Unity.

Project description:

In this Unity project, I developed a first-person shooter zombie horror game, showcasing my skills in coding, game design, and animation integration.

Date:

Summer 2024

Role:

Gameplay designer, AI programmer and gameplay programmer.

Gameplay Mechanics:

  • Implemented a dynamic inventory system for managing resources like weapons and items.
  • Designed an interaction system for opening, closing, and locking doors, as well as picking up items and completing objectives.

    • AI Development:

  • Programmed zombie AI to patrol, detect the player, and launch attacks.
  • Balanced zombie behaviors to create challenging and immersive gameplay.

    • Game Logic:

  • Developed win/loss conditions for player survival and game completion.
  • Integrated systems for player damage, health management, and game progression.

    • Asset Integration:

  • Assembled the game using prebuilt maps and 3D models, enhancing them with custom animation setups and programming.
  • Blended animations for zombies and environmental interactions to achieve a cohesive game experience.
