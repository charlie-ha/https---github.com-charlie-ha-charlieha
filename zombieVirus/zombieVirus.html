Horror Game

Project type:

First Person Horror Survival Game in Unity.

Project description:

In this Unity project, I developed a first-person shooter zombie horror game, showcasing my skills in coding, game design, and animation integration.

-----------

The player boards an island to find a cure for a zombie virus outbreak. Unfortunately, the player is infected and has to race against time before turning into a zombie.

Date:

Summer 2024 to present.

Role:

Gameplay designer, AI programmer and gameplay programmer.

Project highlight:

Game Logic:

Developed win condition and full game completion.

These are timestamps for each milestone in the game:

Developed loss conditions for player survival.

Integrated systems for player damage, health management, and game progression.

Asset Integration:

Created Main menu and loading screen using Tersein Game Easy Loading Screen pack on Unity.

Assembled and programmed the game using prebuilt maps, 3D models, and animations .

Assembled animations for zombies, fps arm and environmental interactions to achieve a cohesive game experience.

Gameplay Mechanics:

Implemented a dynamic inventory system for managing weapon and items resources.

Designed an interaction system for opening, closing, locking doors, as well as picking up items and completing objectives.

AI Development:

Programmed zombie AI to patrol, detect and follow the player, and attacks.

Varied zombie behaviors to create challenging and immersive gameplay.
