CHARLIE HA

Grill-A-Warfare

Project type:

First Person Shooter Cooking-themed Game in Unity.

Team project.

Project description:

The concept is a first person shooter / Doom - like, but everything is cooking themed and very silly and fun instead of gory. The main character is a former soldier and current chef fighting off hordes of mutant cooking ingredients to save the integrity of his kitchen.

Date:

September-December 2024

Role:

Gameplay designer, gameplay programmer.

Project highlight:

Game Logic:

Developed win condition and full game completion.

These are timestamps for each milestone in the game:

Developed loss conditions: The player can die but have infinite respawns. Only after 10 minutes, the player loses and a loss screen pops up. (For demonstration purposes, the time is reduced to 30s)

Integrated systems for player to deal damage, receive damage from enemies, recover health, and progress the game.

Asset Integration:

Created Main menu and loading screen using Tersein Game Easy Loading Screen pack on Unity.

Assembled and programmed the game using maps and 3D models built by my team.

Gameplay Mechanics:

Press keys 1, 2, 3, 4 to change to each weapons fork, Knife Block Gun, Soup Can Grenade Launcher, Melee Rolling Pin .

Designed an interaction system for opening, closing locked doors, as well as picking up keys. Completing objectives: defeat all enemies within the room before time runs out.
